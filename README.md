## 스터디 규칙 

- **진행 형태**
    
    > 한 사람이 다음주 태그 설명(수도코드 및 기본 개념 포함) + 공통문제 준비(백준 실버 2~3 수준)<br/>
    공통 1 + 개인 1 이상 풀이<br/>
    알고리즘 풀이 시 개인적으로 자료를 찾아보는 것은 최대한 지양<br/>
    스터디 진행 시 발표자 발표가 끝난 후 스터디원 모두가 공통문제에 대한 풀이 설명 및 타 스터디원의 피드백
